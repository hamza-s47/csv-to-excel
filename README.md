# CSV to Excel Converter
A simple command-line tool that converts CSV files into Excel (.xlsx) format using Python and Pandas.

## Features:
- Convert any .csv file to .xlsx
- Extremely simple CLI usage
- Uses Pandas for fast and reliable file operations

## Tech Stack:
- Python 3
- Pandas

## Installation:
git clone https://github.com/hamza-s47/csv-to-excel <br/>
cd csv-to-excel <br/>
pip install -r requirements.txt

## Usage (CLI):
python3 main.py /path/to/input.csv /path/to/output.xlsx (Linux/Mac) <br/>
python main.py /path/to/input.csv /path/to/output.xlsx (Windows)
